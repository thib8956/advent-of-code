# Advent of code

My solutions to the [advent of code](https://adventofcode.com/) challenges, written in python

## How to run

### Install project

Run `make install` or

Run from root directory (inside a virtualenv):

```shell
$ pip install -e .
```

### Run a single day

```shell
$ aoc <year> <day>
```

### Run a whole year

```shell
$ aoc <year>
```
